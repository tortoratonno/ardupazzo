
per tutte le librerie che vanno in conflitto:

1.	cancellare tutte le copie della libreria tranne quella versionata
2.	riavviare l'ide
3.	Sketch -> Import Library...
	se lo sketch si trova nella cartella versionata degli sketch
	nella sezione finale dovrebbe comparire la libreria
	in caso contrario:
	... -> Add Library...
	e andarsela a pescare
	

For information on installing libraries, see: http://arduino.cc/en/Guide/Libraries
